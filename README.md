# CSSgen

A free, open-source CSS effects generator that runs entirely in your browser. No installs, no sign-up, no backend.

## Effects

- Box Shadow — multi-layer, inset support, full color + opacity control
- Gradient — linear, radial, and conic with unlimited color stops
- Glassmorphism — blur, saturation, background and border opacity
- Text Shadow — horizontal, vertical, blur, color and opacity
- Border Radius — uniform or individual corners

## Usage

Just open it. Tweak the sliders. Copy the CSS.

Live demo: [tabato.github.io/css-tool](https://tabato.github.io/css-tool)

## Run locally
```bash
git clone https://github.com/tabato/css-tool
cd css-tool
python3 -m http.server 8080
```

Then open `http://localhost:8080`

## Stack

Vanilla HTML, CSS, and JS. No frameworks, no dependencies, no build step.
